# CRUD-Nodejs-Mysql  
# Getting Started

Install Dependencies

```bash
npm install
npm i ejs
npm i express
npm i express-myconnection
npm i morgan
npm i mysql
```

Install devDependencies
```bash 
npm i nodemon
```

Run with command:

```bash
npm run dev
```


## Endpoints

```bash
http://localhost:3000
```
```bash
http://localhost:3000/add
```
```bash
http://localhost:3000/delete:id
```
```bash
http://localhost:3000/update:id
```
